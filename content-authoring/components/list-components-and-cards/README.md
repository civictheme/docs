# List components and cards

### Components

CivicTheme offers two list components out of the box:

* [Automated list ](../../../development/drupal-theme/automated-list.md)
* [Manual list](manual-list.md)

### Cards

There are 12 different cards to choose from to style the list, depending on your content requirements.

Not all cards are available on both lists.

#### Cards available for the Automated list

* Promo card
* Navigation card
* Snippet

#### Cards available for the Manual list

* Event card
* Event reference card
* Navigation card
* Navigation reference card
* Promo card
* Promo reference card
* Publication card
* Service card
* Snippet
* Snippet reference
* Subject card
* Subject reference card
